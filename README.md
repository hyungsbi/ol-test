# 설치
* 방법 1 : git cloen https://github.com/hyungsbi/ol-test.git 실행
> git cloen https://github.com/hyungsbi/ol-test.git
* cloning된 곳으로 이동하여 npm install 실행
> npm install
* 방법 2 : "Download ZIP" 실행, 다운받은 파일 압축 풀기
* 압축풀린 곳으로 이동 후 npm install 실행
> npm install
